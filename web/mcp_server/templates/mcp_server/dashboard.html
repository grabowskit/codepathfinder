How to Setup Claude Desktop

  1. Open Claude Desktop config file. ~/Library/Application Support/Claude/claude_desktop_config.json
  2. Add the following configuration:
{
  "mcpServers": {
    "codepathfinder": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-sse-client"],
      "env": {
         "MCP_SSE_URL": "{{ mcp_url }}",
         "OAUTH_CLIENT_ID": "{{ client_id }}",
         "OAUTH_CLIENT_SECRET": "YOUR_CLIENT_SECRET" 
      }
    }
  }
}

Note: Replace YOUR_CLIENT_SECRET with the value from above.

Note: The official Claude Desktop currently does not natively support OAuth flows. It expects STDIO or SSE.
