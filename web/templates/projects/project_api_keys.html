Setup Instructions

Click to expand setup guide for Claude Desktop

Quick Start:

  1. Generate an API key below if you haven't already
  2. Copy your API key when it appears (you'll only see it once!)
  3. Scroll down to the "Claude Desktop Configuration" section
  4. Copy the configuration snippet
  5. Open your Claude Desktop config file:
    • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
    • Windows: %APPDATA%/Claude/claude_desktop_config.json
  6. Add or merge the configuration into your config file
  7. Replace YOUR_API_KEY_HERE with your actual API key
  8. Save the file and restart Claude Desktop
  9. The MCP server will appear as "codepathfinder-{{ project.name|lower|slugify }}" in Claude Desktop

Note: Once configured, Claude Desktop will be able to search your indexed codebase using 6 powerful tools: semantic search, symbol mapping, file reading, and more.
